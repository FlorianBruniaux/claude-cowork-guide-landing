7 Golden Rules

Follow these to avoid the most common mistakes

1

Always review the plan

Cowork proposes, you decide. Read every execution plan before approving. Look for unexpected actions.

2

Dedicated workspace only

Never grant access to ~/Documents, ~/Desktop, or your home folder. Create ~/Cowork-Workspace/ instead.

3

No credentials in workspace

Keep passwords, API keys, and secrets out of Cowork-accessible folders. Content is sent to Anthropic servers.

4

Verify file sources

Prompt injection risk in untrusted files. Malicious PDFs can manipulate Cowork behavior.

5

Budget error correction time

OCR accuracy is ~97% for fields, ~63% for line items. Always verify extracted data.

6

Disconnect VPN

The #1 reported issue. VPN breaks VM networking. Disconnect before using Cowork.

7

Backup before destructive ops

No undo for file deletions. Run cp -R ~/Cowork-Workspace/ ~/Cowork-Backup/ first.

Frequently Asked Questions

Quick answers to common questions

What is Cowork?

Cowork is Claude's agentic desktop feature that manipulates local files, creates documents, and organizes your workspace‚Äîwithout writing code. Think of it as Claude Code for knowledge workers.

How much does Cowork cost?

Cowork requires a Pro ($20/month) or Max ($100-200/month) subscription. Pro is recommended for light use only‚Äîquota exhausts in ~1-1.5 hours of intensive use.

Does Cowork work on Windows or Linux?

Not yet (January 2026). macOS only. Windows is on Anthropic's roadmap but has no ETA. Linux has no official announcement. Use Claude Code for Linux agentic capabilities.

Can I use Cowork with a VPN?

No. VPN software creates routing conflicts with Cowork's internal VM networking. This is the #1 reported issue. Disconnect VPN completely before using Cowork.

Is Cowork secure?

There's no official security documentation yet (research preview). Best practices: use dedicated workspace folder, review every plan, keep credentials out, verify file sources.

What can Cowork NOT do?

Cowork cannot: execute code/scripts, make API calls, access cloud storage directly (Google Drive, Dropbox), process audio/video, decrypt encrypted files.

When should I use Chat vs Cowork?

Chat (80%): reasoning, writing, coding, exploratory tasks. Cowork (20%): batch file operations, document creation, browser automation. Together they enable workflows impossible with either alone.

What's the difference from Claude Code?

Claude Code: terminal interface, full shell access, for developers. Cowork: desktop app, file-only access, for knowledge workers. They share architecture but differ in capabilities.

Can Cowork create Excel formulas?

Yes! Cowork can create Excel files with working formulas, multiple sheets, and formatting. Specify your regional setting (US comma vs EU semicolon) in your prompt.

What are the usage limits?

Usage resets every 5 hours (not daily/monthly). Pro tier: ~45 short messages per reset (~1-1.5h intensive use). Max tier: 5x-20x more. File processing consumes tokens rapidly.
